Tags de Títulos

As tags de titulos são as de "h1" até "h6", definido o nível de hierarquia e relevância entre elas, sendo que a "h1" é a de maior nível entre eles e tamanho, vejo no exemplo abaixo:

  • TAG TÍTULO H1

  • TAG TÍTULO H2

  • TAG TÍTULO H3

  • TAG TÍTULO H4

  • TAG TÍTULO H5
  • TAG TÍTULO H6

Tags de Parágrafo

A tag de parágrafo é a tag "p", que são usadas para apresentarem textos como esse no conteúdo da página.

Tags de marcações e formatação

Algumas das tags de marcação de estilização de texto são a "mark", que traz uma marcação ao texto dessa forma, a tag "i" que deixa o texto em itálico, a tag "strong", que faz com que o texto fique em negrito, a tag "u", que faz com que o texto fique sublinhado, e a tag "del", que faz com que o texto fique com um tracejado no meio.

Tags <"sup> e <"sub>*

As tags "sup" e "sub", fazem com que o valor/texto anunciado entre eles, fique elevado ou um pouco abaixo do texto próximo respectivamente, confira no exemplo abaixo:

texto super-legaltag sup

texto super-legaltag sub

Tags de Lista

As tags de lista são respectivamente "ol" para listas organizadas em que cada item da lista serão enumeradas e "ul" para listas desargonizadas, onde apenas marcadores ficarão na frente de cada item da lista, confira no exemplo abaixo:

Tag <"ol>*

  1. Item 1
  2. Item 2
  3. Item 3

Tag <"ul>*

  • Item 1
  • Item 2
  • Item 3

A tag fonte server para trazer formatação ao texto, alterando seu tamanho, cor e o tipo de fonte, confira no exemplo abaixo:
